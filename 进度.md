### auto_explorer v12 当前架构

```
传感器管线:
  /robot/lidar (Gazebo原始, 3600采样, 10Hz)
    → cloud_to_scan v5 → /scan (710/720rays, robot/base_link帧)
      ├─→ slam_toolbox → /map → 前沿检测
      └─→ auto_explorer 直接读 /scan → APF势场避障

状态机 (优先级从高到低):
  1. HARD BRAKE: 前方<1.0m → 后退1.2s + 转弯(1s后退+2s前进)
     → 连续刹车=LOOP, ≥2次触发反向U-turn
     → 标记当前位置为 _vst 禁区
  2. ACQUIRE: YOLO检测到目标但无3D → 暂停+慢转等fusion
  3. →CHASE: 有检测有3D → APF追踪
  4. SPIN_WAIT: 旋转1.2圈+map>1%, 超时20s
  5. EXPLORE: 卷积找前沿→评分→A*→LOS→APF驱动
  6. no-frontier<20: 前半球±120°找最开阔方向推, 避开_bias_angle
  7. no-frontier≥20: 原地旋转扫描建图(lin=0,ang=0.6), 每10s清_vst
  8. stuck: 5s→esc+1→3s转弯; esc=2→随机方向; esc=3→标记禁区+8s长撤退
  9. fallback: 朝前推, 避_bias_angle

避障: APF (CoNi-OA 启发)
  排斥力(局部帧) → rotation matrix → 世界帧 + 吸引力(世界帧)
  D_SAFE=2.5m, D_BACK=1.0m, 最近距离=0.5m, 前半球排斥
  坐标系旋转修复: 局部排斥力转世界帧后再与引力相加
```

### 前沿探索评分

```
score = 2.0 × log(size)/log(100) - 2.0 × (dist/15) - 1.5 × (heading_err/π)
  min_cluster=15像素 (~0.75m²), 3m马赛克visited去重
```
