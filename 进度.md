### auto_explorer v11 当前架构

```
传感器管线:
  /robot/lidar (Gazebo原始, ~10Hz, 900pt×16线, lidar_link局部帧)
    ├─→ cloud_to_scan v5 → /scan (720rays, robot/base_link帧) → slam_toolbox → /map
    └─→ auto_explorer 内置 _cloud_to_scan_ranges → APF势场避障

状态机 (优先级从高到低):
  1. HARD BRAKE: 前方<1.0m → 锁死后退+转向, F>3m解锁
  2. ACQUIRE: YOLO检测到目标但无3D → 暂停(lin=0)+慢转等fusion同步
  3. →CHASE: 有检测有3D → APF追踪; 丢失4s→回EXPLORE
  4. SPIN_WAIT: 原地旋转建图(1.2圈+map>1%), 超时20s
  5. EXPLORE: /map卷积找前沿→增益/距离/朝向评分→A*→LOS→APF驱动
  6. no-frontier: 前半球±120°搜最开阔方向推
  7. fallback: 朝前进方向推3m

避障: 人工势场 APF (借鉴 CoNi-OA)
  排斥力(机器人局部帧) → 旋转矩阵→世界帧 + 吸引力(世界帧) → 合力→cmd_vel
  仅排斥前半球(±135°), 背后无视
  D_SAFE=2.5m, D_BACK=1.0m, 最近有效距离=0.5m
```

### 前沿探索评分

```
score = 2.0 × log(size)/log(100) - 2.0 × (dist/15) - 1.5 × (heading_err/π)

  优先选: 前方、足够大、不太远的边界
  min_cluster=5像素 (~0.25m²)
  3m马赛克visited去重
```
