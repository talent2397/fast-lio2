### auto_explorer v10 当前架构

```
传感器管线:
  /robot/lidar (Gazebo原始, ~10Hz, 900pt×16线)
    ├─→ cloud_to_scan v3 → /scan (720rays, robot/base_link帧) → slam_toolbox → /map
    └─→ auto_explorer 内置 APF (720pts逐个算排斥力)

状态机:
  SPIN_WAIT (1.2圈, 等map>1% free)
    → EXPLORE (/map→卷积找前沿→log(gain)-w×dist-w×heading评分→A*→LOS→APF驱动)
    → CHASE (YOLO检测→fusion 3D坐标→APF追上去)
    → REACHED (目标2m内停车)

避障: 人工势场 APF (借鉴 CoNi-OA 调制矩阵)
  每LiDAR点(d,θ) → 排斥力 F_rep = -k/d²·(cosθ, sinθ)
  目标方向     → 吸引力 F_att = k_att
  合力 → cmd_vel (无阈值, 纯反应式, 每scan点独立贡献)
```

### 前沿探索评分 (借鉴 explore_lite + FUEL)

```
score = 2.0 × log(size) - 2.0 × (dist/15) - 1.5 × (heading_err/π)

  优先选: 前方、足够大、不太远的边界
  到达后:  标记 3m 马赛克 visited → 防止回头
  全部访问: 清空 visited 重搜 → 地图扩展后新边界出现
```
