### 架构演变 (Phase 2.7)

```
放弃 Nav2: lifecycle_manager 不自动激活 + collision_monitor 缺少配置
        + 插件名 / 与 :: 混用 + map 帧不存在导致 planner 拒绝配置
        → 总共遇到了 8+ 个阻塞级 bug

自研替代方案演进:
  v1: 两节点 (explore_coordinator + simple_navigator)
       → 死锁: 无目标不动, 不动无地图
  v2: smart_navigator (A* + 视线平滑)
       → 无目标直冲撞墙 + SPIN_WAIT 后地图未就绪
  v3: 单节点 auto_explorer (边建图边探索)
       → 碰撞检测弱 + 重复探索同一区域 + fusion 向量化性能问题
  v4: 精简版 auto_explorer (当前) ← 调优中
```

### auto_explorer 当前架构

```
SPIN_WAIT (1.2圈建图) → EXPLORE (读/map找前沿→A*路径→纯追踪)
                             ↓ (YOLO检测到目标)
                        CHASE (追fusion 3D坐标→A*→到达)
                             ↓
                        REACHED (停车, 任务完成)
```
